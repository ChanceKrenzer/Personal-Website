Skills

What I Know

CAD
ICON

Computer-Aided Design

My primary design environment is SolidWorks, where I work almost exclusively in parametric modeling — building parts driven by global variables so designs can be rapidly iterated without breaking assemblies. I've designed everything from drone frames and motor mounts to payload release mechanisms and precision fabrication jigs. I also use PDM for version control on team projects and am comfortable working across part, assembly, and drawing workflows.

SIM
ICON

Simulation & Analysis

I use simulation as a core part of my design loop, not an afterthought. In SolidWorks FEA I validate structural designs under real load cases — for example, verifying drone arms can withstand motor-induced forces while keeping weight minimal. For aerodynamics, I've run SolidWorks Flow Simulation to iterate on frame geometry for drag reduction on high-speed builds. I've also worked with ANSYS on [your ANSYS context here].

FAB
ICON

Fabrication & Manufacturing

I'm comfortable taking a design from a digital file to a finished physical part. I regularly use a CNC waterjet for precision metal cutting, a mill and lathe for turned and milled aluminum components, and both MIG and TIG welding for structural joints. On the additive side, I use FDM 3D printing (primarily PETG and PLA) for structural prototypes and final parts. I also work with laser cutters and do fine-pitch soldering for electronics work.

CODE
ICON

Programming

I genuinely enjoy coding — there's something satisfying about writing logic that talks directly to hardware. My strongest languages are Python for scripting, data processing, and automation; Java for structured, object-oriented work; and MATLAB for simulation and numerical analysis. I've also written low-level C for embedded systems and flight control software in Arduino, including a hand-tuned PID controller for a custom drone.
